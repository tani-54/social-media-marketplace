# Social Media Marketplace

A platform where brands and influencers can connect and collaborate.  
This project allows users to create listings, explore influencers, and manage their marketplace activities.

## Features
- User authentication using Clerk
- Influencer marketplace listings
- Featured listings
- Messaging system (planned)
- Premium subscription plans
- Modern responsive UI

## Tech Stack
- React
- Vite
- Tailwind CSS
- Clerk Authentication
- Redux Toolkit

## Installation

Clone the repository:

git clone https://github.com/tani-54/Social-media-marketplace.git

Go to project folder:

cd Social-media-marketplace/client

Install dependencies:

npm install

Run the project:

npm run dev
